Create Visitor:
```php
$visitor = Visitor::create([
    'first_name' => 'John',
    'last_name' => 'Doe',
    'phone' => '+1234567890',
    'email' => 'john@example.com',
    'visit_date' => now(),
    'service_attended' => 'Sunday Service',
]);

// Generate QR code
$visitor->generateQRCode();

// Send welcome
$visitor->sendWelcomeMessage('both');

// Track milestone
$visitor->addJourneyMilestone('first_visit', 'First visit');
```

Record Attendance:
```php
$visitor->recordAttendance('Sunday Service', 'qr');
```

Log Follow-Up:
```php
$visitor->logFollowup(
    'phone',
    'Great conversation!',
    'interested',
    now()->addWeek()
);
```

Get Analytics:
```php
$stats = [
    'total' => Visitor::count(),
    'converted' => Visitor::converted()->count(),
    'pending' => Visitor::pendingFollowup()->count(),
];
```
