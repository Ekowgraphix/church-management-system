===============================================
HOW TO ADD CHURCH BACKGROUND IMAGE
===============================================

STEP 1: Save the neon cross church image

STEP 2: Rename it to exactly:
        church-background.jpg

STEP 3: Place it in THIS folder:
        f:\xampp\htdocs\churchmang\public\images\

        (You are already in the correct folder!)

STEP 4: Verify by visiting:
        http://127.0.0.1:8000/images/church-background.jpg

STEP 5: Visit the logout page:
        http://127.0.0.1:8000/logout

You should see the beautiful church background!

FILE LOCATION REQUIRED:
f:\xampp\htdocs\churchmang\public\images\church-background.jpg
