📝 SAMPLE CONTROLLER METHODS
═══════════════════════════════════════════════════════════════

FINANCE CONTROLLER:
```php
public function dashboard()
{
    $stats = [
        'total_income' => Finance::where('type', 'Income')->sum('amount'),
        'total_expenses' => Finance::where('type', 'Expense')->sum('amount'),
        'income_change' => 12,
        'expense_change' => 8,
        'pending_count' => 5,
    ];
    
    $transactions = Finance::latest()->limit(20)->get();
    
    return view('finance.dashboard', compact('stats', 'transactions'));
}
```

VISITORS CONTROLLER:
```php
public function dashboard()
{
    $stats = [
        'visitors_this_week' => Visitor::whereBetween('visit_date', [now()->startOfWeek(), now()->endOfWeek()])->count(),
        'converted_members' => Visitor::where('conversion_status', 'converted')->count(),
        'pending_followup' => Visitor::where('followup_status', 'Pending')->count(),
        'returning_visitors' => Visitor::where('visit_count', '>', 1)->count(),
        'visitor_change' => 15,
        'conversion_rate' => 35,
    ];
    
    $visitors = Visitor::latest()->limit(20)->get();
    
    return view('visitors.dashboard', compact('stats', 'visitors'));
}
```
