💡 HOW TO USE SETTINGS NOW
═══════════════════════════════════════════════════════════════

Even without the fancy UI, you can use settings:

In your controllers/views:
```php
use App\Models\Setting;

// Store
Setting::set('church_name', 'Grace Church');
Setting::set('church_email', 'info@church.org');

// Retrieve
$churchName = Setting::get('church_name');
```

In Blade views:
```blade
{{ Setting::get('church_name', 'Church Name') }}
```
